# Recherche de produit - Service Web
Ce service web permet de rechercher des produits en fonction de différents critères tels que le mot-clé de recherche, la catégorie, le fabricant et le prix. Il fournit également des informations détaillées sur un produit donné.

## Fonctionnalités
searchProducts: cette opération permet de rechercher des produits en fonction de différents critères tels que le mot-clé de recherche, la catégorie, le fabricant et le prix. Elle renvoie une liste de produits correspondants.

getProductDetails: cette opération permet d'obtenir des informations détaillées sur un produit donné en fournissant son identifiant. Elle renvoie toutes les informations disponibles sur le produit telles que son nom, sa description, son prix, sa disponibilité et son fabricant.

## Utilisation
Ce service web est accessible via une interface SOAP. Vous pouvez utiliser votre client SOAP préféré pour accéder aux opérations fournies par le service web.
L'URL du service est http://example.com/productSearchService. Vous devez configurer votre client SOAP pour accéder à cette URL.
